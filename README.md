# اختبار RTU8

هذه صفحة ويب ثابتة لا تحتاج إلى أي مكتبات أو خادم خاص لتعمل.

## التشغيل محليًا

يمكنك فتح الملف مباشرةً باستخدام أي متصفح:

- افتح [index.html](index.html)

أو شغّل خادمًا محليًا بسيطًا من المجلد:

```bash
python -m http.server 8000
```

ثم افتح:

```text
http://127.0.0.1:8000/
```

## ما الذي يحتويه المشروع

- صفحة اختبار شخصية عربية
- 10 أسئلة
- نتيجة شخصية مبنية على أعلى سمة مختارة
- زر إعادة البدء
